## [RU] Визуализация сортировки 
Программа позволяет отобразить сортировку на изображении, путём переставления пикселей. 
Но с некоторыми ограничениями.
* Размер картинки `~2000` пикселей (можно больше, но будет долго работать)
* Картинка в `.png`
* Сортировка содержит только базовые операции
  * Обращение по индексу
  * Установка по индексу
  * Длинна списка

Также доступны настройки в `config.py`.

### Необходимые библиотеки:
```shell
pip install opencv-python
pip install numpy
pip install Pillow
```

## [EN] Sorting visualization
The program allows you to display sorting on the image by rearranging the pixels.
But with some restrictions.
* Image size `~2000` pixels (more is possible, but it will work for a long time)
* Picture in `.png`
* Sorting contains only basic operations
   * Reference by index
   * Installation by index
   * List length

Settings are also available in `config.py`.

### Required libraries:
```shell
pip install opencv-python
pip install numpy
pip install Pillow
```
